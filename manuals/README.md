# 中文手册目录

这个文件夹只保存学习和操作手册，不参与 PlatformIO 编译，也不会被烧录进 ESP32 固件。

| 手册 | 内容 |
|---|---|
| [CODE_WALKTHROUGH_ZH.md](CODE_WALKTHROUGH_ZH.md) | 项目结构、运行流程、各代码文件作用说明。 |
| [TASK_COMPLETION_GUIDE_ZH.md](TASK_COMPLETION_GUIDE_ZH.md) | 两个核心任务的完成步骤和可参考代码。 |
| [CPP_QUICK_GUIDE_ZH.md](CPP_QUICK_GUIDE_ZH.md) | 面向 Python 学习者的 C++ / Arduino 快速参考。 |

PlatformIO 默认只会编译 `src/`、项目依赖库和相关配置文件。这里的 `.md` 文档不会进入固件镜像。
